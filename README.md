# code-switching-nlp-analysis-bangla-english
This repository contains the code and dataset used for NLP analysis of Bengali-English code-mixed social media data.
## Files
- `1st_research.py` → Main Python script for analysis  
- `dataset_zidan-research_2026-02-15_10-06-48-661 (3).json` → Primary dataset (large file; use "View Raw" to access)  
- `Master_Research_Final_Table (1).xlsx` → Processed dataset (download by clicking "View Raw" and open in Excel or Google Sheets)  
## Note
Due to GitHub limitations, large JSON and Excel files cannot be previewed directly in the browser.
